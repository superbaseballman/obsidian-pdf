# Changelog

All notable changes to the PDF Editor plugin will be documented in this file.

## [1.0.1] - 2026-06-29

### Bug Fixes

#### Critical Fixes
- **内存泄漏**：`loadFile` 加载新 PDF 前未清理旧文档和渲染任务，导致内存持续增长
- **注释加载后不显示**：`loadAnnotations()` 读取了 JSON 数据但从不渲染到页面，重新打开 PDF 后所有注释消失
- **缩放后注释消失**：`setZoom` 重新渲染页面时清空了所有注释 DOM，现在缩放后会自动重新渲染所有已保存注释
- **渲染竞态条件**：`renderAllPages` 不取消之前的渲染任务，快速缩放时产生竞态，现在会先取消所有进行中的任务
- **高亮功能完全失效**：文本层 `pointer-events: none` 导致无法选中文本，新增 `.selectable` 状态，高亮工具激活时启用文本选择
- **高亮颜色透明度拼接错误**：`${color}40` 对 3 位 hex（如 `#FF0`）会生成无效颜色，新增 `hexToRgba()` 正确处理 3/6 位 hex
- **橡皮擦擦除不持久化**：橡皮擦笔画从未保存到 JSON，缩放后擦除效果消失，现在保存橡皮擦笔画并在重渲染时用 `destination-out` 重新应用

#### General Fixes
- **工具栏设置不持久化**：颜色选择器、画笔/橡皮擦大小滑块修改后不调用 `saveSettings()`，重启后丢失
- **页码输入框无响应**：输入页码后按 Enter 无法跳转，添加 `change`/`Enter` 事件监听
- **缩放下拉框不同步**：用快捷键或适应宽度改变缩放后下拉框不更新，新增 `updateZoomControl()`
- **`fitToWidth` 硬编码**：假设页面宽度为 612px（letter 尺寸），对 A4 等其他尺寸不准，改为读取实际第一页宽度
- **`updateStatusbar` 不可靠**：用 `lastChild` 定位缩放元素，易受其他子元素影响，改用 class 选择器
- **大纲不递归**：只渲染顶层大纲项，嵌套子项丢失，新增 `renderOutlineItems()` 递归渲染
- **笔按钮逻辑错误**：`pointerup` 时用 `e.buttons` 掩码判断（此时已不包含释放的按钮），导致工具无法恢复，改为用追踪的状态变量
- **绘图首点不绘制**：单击不留痕迹，只有移动才出现线条，现在首点绘制一个圆点
- **线宽更新滞后**：`handlePointerMove` 先画线再更新线宽，导致首段用旧宽度，调整更新顺序
- **`onClose` 丢失未保存注释**：关闭视图前不保存，现在关闭前调用 `saveAnnotations()`
- **`file-open` 事件循环**：每次打开 PDF 都触发重定向，干扰用户切换标签页，增加活跃视图判断
- **`openPdf` 重复调用注释加载**：`loadFile` 已加载注释，`openPdf` 又调用一次，移除冗余调用
- **`openPdf` 堆积空 leaf**：每次打开新 PDF 都创建新标签页，改为复用已有的空编辑器 leaf
- **`main.ts` 类型错误**：`file-menu` 事件签名不匹配，使用正确的 `TAbstractFile` 类型

### UX Improvements
- **加载指示器**：加载 PDF 时显示旋转加载动画和提示文字，避免空白等待
- **键盘快捷键**：支持 `←/→`、`PageUp/PageDown` 翻页，`Home/End` 跳首尾页，`+/-` 缩放，`0` 重置缩放
- **缩放保持滚动位置**：缩放后按比例恢复滚动位置，视图不会跳到顶部
- **侧边栏状态持久化**：切换侧边栏后保存到设置，下次打开保持
- **注释右键删除**：高亮和备注支持右键删除
- **备注图标跨平台一致**：用 CSS 绘制图标替代 emoji `📝`，避免不同平台渲染差异
- **错误提示更详细**：加载失败时显示具体错误信息
- **空选择保护**：高亮工具点击但不选中文本时不再创建零尺寸的高亮块
- **缩放浮点精度**：`setZoom` 四舍五入到 0.01，避免浮点漂移导致无限重渲染

- 工具栏与侧边栏
- 橡皮擦工具
- 页面缩放与导航
- 大纲/目录导航
